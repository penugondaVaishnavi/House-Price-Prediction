# House Price Prediction

This project is part of the Alfido Tech Machine Learning Internship.

## Objective
To predict house prices using Machine Learning regression techniques.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Algorithm Used
- Linear Regression

## Project Workflow
1. Load housing dataset
2. Analyze and preprocess data
3. Visualize relationships
4. Train regression model
5. Evaluate model performance

## Result
The model successfully predicted house prices using Linear Regression.
